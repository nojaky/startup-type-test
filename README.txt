나는 창업형 인간일까, 직장형 인간일까? - 웹 테스트

구성 파일
- index.html : 메인 페이지
- style.css : 디자인
- script.js : 질문, 점수 계산, 결과 출력


특징
- 15문항 테스트
- 5개 축 분석: 현실 감각, 실행력, 멘탈 내구도, 반복 운영력, 시스템화 능력
- 진행률 % 표시
- 결과 유형 6가지
- 창업 적합도 원형 게이지
- 5개 축 막대그래프
- 결과 공유 버튼
- 추천 버튼 2개

참고
이 테스트는 재미형 성향 테스트이며, 실제 창업 성공 가능성을 보장하거나 진단하는 도구가 아닙니다.
