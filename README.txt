나는 창업형 인간일까, 직장형 인간일까? - 웹 테스트

구성 파일
- index.html : 메인 페이지
- style.css : 디자인
- script.js : 질문, 점수 계산, 결과 출력

배포 방법
1. 압축을 풀고 index.html, style.css, script.js 파일을 같은 폴더에 업로드합니다.
2. GitHub Pages, Cloudflare Pages, Netlify 같은 정적 호스팅에 그대로 배포할 수 있습니다.

추천 버튼 링크 수정
script.js 상단의 아래 두 줄을 실제 주소로 바꾸면 됩니다.

const SURVIVAL_URL = "#";
const COST_SIMULATOR_URL = "#";

예시
const SURVIVAL_URL = "https://1985coffee.com/tools/survival/";
const COST_SIMULATOR_URL = "https://1985coffee.com/tools/cafe-cost/";

특징
- 15문항 테스트
- 5개 축 분석: 현실 감각, 실행력, 멘탈 내구도, 반복 운영력, 시스템화 능력
- 진행률 % 표시
- 결과 유형 6가지
- 창업 적합도 원형 게이지
- 5개 축 막대그래프
- 결과 공유 버튼
- 추천 버튼 2개

참고
이 테스트는 재미형 성향 테스트이며, 실제 창업 성공 가능성을 보장하거나 진단하는 도구가 아닙니다.


[2026-07-01 수정]
- 창업 성향 점수 계산식을 보정했습니다.
- 기존에는 보조 점수까지 총점에 합산되어 일부 조합에서 100%가 과대 표시될 수 있었습니다.
- 수정 후에는 5개 축의 실제 가능한 최대점수 기준으로 각 축을 계산하고, 전체 점수는 축 평균과 가장 낮은 축을 함께 반영합니다.
- 결과 화면의 문구를 “창업 적합도”에서 “창업 성향 점수”로 조정했습니다.

[v3 공유 링크 업데이트]
- 결과 공유하기 버튼이 단순 페이지 주소가 아니라, 결과 유형/창업 성향 점수/5개 축 점수를 URL 해시에 담아 공유합니다.
- 공유받은 사용자가 링크로 접속하면 새 테스트 화면이 아니라 공유된 결과 화면이 바로 표시됩니다.
- URL 해시는 브라우저 내부에서만 해석되며 별도 서버 저장 없이 동작합니다.
